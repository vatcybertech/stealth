# Vermillion Axis Technologies

Marketing site for a premium custom software development firm.

## Tech Stack

Next.js 14 | Tailwind CSS | Framer Motion | TypeScript

## Quick Start

```bash
git clone <repo-url>
cd vermillion-site
npm install
npm run dev
```

## Build

```bash
cd vermillion-site
npm run build
```

Output is written to `vermillion-site/out/`.
